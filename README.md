# Book Management Website

A full-stack web application built using the MERN stack for browsing and managing books.

## Features
- View and browse books  
- Add and manage book data  
- REST API integration  
- Responsive user interface  

## Tech Stack
- Frontend: React.js  
- Backend: Node.js, Express.js  
- Database: MongoDB  

## Live Demo
[https://bookwebsite-g8rv-lhf9shulc-itamii5823s-projects.vercel.app/](https://bookwebsite-eta.vercel.app/)

## Installation
git clone https://github.com/itamii5823/bookwebsite  
cd bookwebsite  
npm install  
npm start
